# 搜索与检索任务集

本目录包含用于评估搜索和检索系统的任务定义。

## 任务分类

| 类别 | 任务 | 描述 |
|------|------|------|
| **文本问答** | SQuAD 2.0, CMRC 2018 | 阅读理解与抽取式问答 |
| **多模态** | ChartQA, COCO 检索, AudioCaps | 跨模态检索与理解 |
| **领域专用** | FinQA, CUAD, Spider | 金融、法律、SQL 生成 |
| **RAG** | MultiHop-RAG, ELI5 | 检索增强生成 |
| **对话** | WildChat, Discord | 对话检索与分析 |
| **代码** | GitHub Issues | 代码相关问题检索 |
| **视频** | MSVD | 视频文本检索 |
| **邮件** | Enron | 邮件搜索与分类 |

## 任务格式

每个任务包含：
- **任务描述**：需要完成的目标
- **数据集信息**：数据来源和结构
- **评估目标**：目标指标和成功标准
- **评估方法**：如何衡量性能

## 快速开始

1. 从上面列表中选择一个任务
2. 阅读 `tasks/<任务名>.md` 中的任务定义
3. 从 `datasets/` 加载对应数据集
4. 实现你的解决方案
5. 使用指定的指标进行评估

## 评估类型

| 类型 | 描述 | 适用任务 |
|------|------|----------|
| **精确匹配** | 答案必须与标准答案完全一致 | SQuAD, CMRC |
| **检索指标** | R@K, MRR, MAP | COCO, AudioCaps, MSVD |
| **执行准确率** | 生成的代码/SQL 产生正确结果 | Spider, FinQA |
| **LLM 评判** | 使用 LLM 评估质量 | WildChat, Discord |
| **F1/ROUGE** | 词级重叠指标 | ELI5, MultiHop-RAG |
