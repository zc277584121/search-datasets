### 一、日常生活类（底库为个人/家庭/本地固定资源/信息）
1. **个人文件/资料检索**
    底库：电脑/云盘里的文档、照片、视频、简历、证件扫描件
    检索：搜「2025年旅行照片」「入职简历最新版」「租房合同扫描件」「考研笔记-高数」
    应用：从海量个人文件中快速定位，不用逐文件夹翻找
2. **家庭物资/账单检索**
    底库：家庭物资台账（米面油/洗护用品库存）、手机支付账单/发票、水电煤缴费记录
    检索：搜「2026年1月超市发票」「家里洗衣液库存」「电费缴费记录-近3个月」「网购牛奶订单」
    应用：核对消费、管理家庭物资，避免重复采购/漏查账单
3. **本地生活资源检索**
    底库：自己整理的本地美食/探店清单、周边便民资源（药店/快递点/维修点）、亲子/遛娃打卡库
    检索：搜「家门口3公里川菜馆」「24小时药店」「周末遛娃室内场地」「宠物医院-性价比」
    应用：基于自己筛选的优质底库，快速找符合需求的本地资源，比通用搜索更精准
4. **收藏内容检索**
    底库：浏览器收藏夹、小红书/抖音收藏的笔记、公众号收藏的文章、知乎收藏的回答
    检索：搜「收藏的电饭煲食谱」「租房改造装修笔记」「职场沟通技巧文章」「宝宝辅食做法」
    应用：从自己标记的优质内容底库中检索，避免重新刷平台找资料

### 二、职场办公类（底库为公司/部门/个人工作专属资源，核心提效）
#### （通用行政/运营）
1. **公司制度/文档检索**
    底库：公司内网的员工手册、行政制度、审批流程、办公指引、会议纪要库
    检索：搜「加班调休制度」「报销审批流程」「部门周会纪要-1月」「办公用品申领规则」
    应用：新员工/老员工快速查公司规则，不用反复问行政/领导
2. **工作模板/素材检索**
    底库：部门共享的PPT模板、Excel报表模板、邮件模板、海报素材、文案库
    检索：搜「商务汇报PPT模板」「市场调研Excel表」「客户催款邮件模板」「节日海报素材」
    应用：从标准化底库中调取素材，不用重复制作，提升工作效率

#### （技术/研发岗）
3. **代码/技术文档检索**
    底库：公司代码仓库、技术文档库（接口文档、部署手册、问题排查笔记）、开源代码收藏库
    检索：搜「用户登录模块代码」「MySQL优化部署手册」「接口文档-支付功能」「Python爬虫封装代码」
    应用：研发人员从底库中调取历史代码/文档，避免重复开发、快速排查技术问题
4. **故障/解决方案检索**
    底库：部门搭建的故障排查台账、问题解决方案库（如服务器报错、工具使用问题）
    检索：搜「Docker容器启动失败解决方案」「Excel卡死修复方法」「Git冲突解决步骤」
    应用：遇到问题时，从历史解决方案底库中快速匹配，不用重新查全网/问同事

#### （市场/销售/客服岗）
5. **客户/资源检索**
    底库：客户档案库（客户信息、沟通记录、需求）、销售渠道库、合作方资源库
    检索：搜「北京-电商行业客户」「2025年合作的供应商」「客户A的需求-定制化开发」「高意向客户名单」
    应用：销售快速定位目标客户、调取客户历史信息；市场快速匹配合作资源
6. **话术/案例检索**
    底库：客服应答话术库、销售谈单案例库、客户投诉处理案例库
    检索：搜「客户说价格高的应答话术」「电商行业谈单案例」「售后退款投诉处理方法」
    应用：客服/销售从标准化底库中调取话术/案例，快速应对客户问题，保证沟通专业性

### 三、学习备考类（底库为个人/机构搭建的学习专属资源）
1. **学习资料/题库检索**
    底库：自己整理的考研/考证资料、刷题题库、网课笔记、课件PPT
    检索：搜「考研英语阅读真题-2025」「教资科二简答题考点」「Python网课笔记-函数」

2. **知识点/错题检索**
    底库：个人错题本（纸质/电子）、知识点总结库、公式定理库
    检索：搜「高数洛必达法则错题」「英语四六级高频词汇-动词」「初中物理浮力公式」
    应用：查漏补缺，快速定位薄弱知识点和错题，高效复习

### 四、内容创作/运营类（底库为创作者/运营搭建的内容资源库）
1. **自媒体内容检索**
    底库：自己的文案素材库、选题库、图片/视频素材库、爆款笔记库
    检索：搜「小红书美食选题」「短视频口播文案模板」「秋季穿搭图片素材」
    应用：自媒体创作者从底库中调取素材，快速创作内容，避免选题/素材荒
2. **电商运营检索**
    底库：电商店铺的商品信息库、标题关键词库、评价回复话术库、活动策划方案库
    检索：搜「卫衣商品标题关键词」「客户差评回复话术」「618店铺活动策划方案」
    应用：电商运营快速匹配商品关键词、应对评价、调取活动方案，提升运营效率

### 五、企业/团队管理类（底库为企业核心资产库）
1. **企业档案/资产检索**

    应用：项目负责人/团队成员快速跟进项目进度，调取项目相关资料

### 七、多模态文档检索类（底库为混合图文表格的复杂文档）
1. **PPT/幻灯片检索**
    底库：公司/个人的PPT文档库、课件库、培训材料库
    检索：搜「Q3销售数据图表」「产品架构图在哪个PPT」「去年年会的演讲稿」
    应用：从大量幻灯片中定位特定图表、文字、布局元素
2. **PDF跨页/跨文档检索**
    底库：研究报告、产品手册、法律文书、学术论文PDF
    检索：搜「合同中的付款条款」「报告里关于市场份额的图表」「论文中的实验方法部分」
    应用：在长PDF中精准定位页面级或布局级内容（表格、图片、段落）
3. **Excel/表格数据检索（NL2SQL）**
    底库：财务报表、数据分析表、库存清单、客户数据表
    检索：搜「2025年销售额超过100万的客户」「库存低于安全线的商品」「上个月新增的订单」
    应用：用自然语言查询结构化表格数据，无需写SQL
4. **扫描文档/OCR检索**
    底库：扫描的合同、发票、证件、手写笔记
    检索：搜「2024年的增值税发票」「身份证扫描件」「手写的会议记录」
    应用：对扫描件进行OCR后建立索引，支持全文检索

### 八、视觉内容检索类（底库为图片/视频媒体资源）
1. **个人相册/照片检索**
    底库：手机/云相册中的照片（含人脸、场景、物体）
    检索：搜「去年在三亚的照片」「有小狗的照片」「穿红色衣服的全家福」「2024年生日派对」
    应用：按人物、场景、时间、物体等多维度检索个人照片
2. **视频内容检索（Moment Retrieval）**
    底库：个人视频库、监控录像、网课视频、会议录像
    检索：搜「视频里讲到机器学习的部分」「客户演示产品的片段」「孩子第一次走路的瞬间」

    检索：找「与这个视频相似的片段」「重复上传的视频」「被搬运的原创内容」

### 十、智能搜索/RAG场景（底库+智能推理）
1. **基础RAG检索**
    底库：企业知识库、产品文档、FAQ库
    检索：搜「如何申请年假」→ 系统检索相关文档 → LLM生成回答
    应用：基于私有知识库的智能问答，带引用来源
2. **Multi-hop多跳推理检索**
    底库：跨多个文档/数据源的信息
    检索：搜「A公司CEO的母校在哪个城市」（需要：A公司→CEO是谁→CEO的母校→母校所在城市）
    应用：需要多步推理、跨文档整合才能回答的复杂问题
3. **Agentic RAG（工具调用+检索）**
    底库：知识库 + 外部工具/API
    检索：搜「帮我查一下最近的航班并对比价格」→ 调用航班API → 检索历史价格 → 生成对比报告
    应用：结合工具调用和知识检索的复杂任务执行
4. **Deep Research深度研究**
    底库：互联网 + 私有知识库 + 多轮搜索
    检索：搜「写一份关于新能源汽车行业的研究报告」→ 多轮网络搜索 → 信息整合 → 生成报告
    应用：需要多轮搜索、信息整合、报告生成的深度研究任务
5. **知识图谱问答（KGQA）**
    底库：结构化知识图谱（实体+关系）
    检索：搜「张三的同事的老板是谁」→ 图谱遍历 → 返回结果
    应用：基于图结构的复杂关系查询

### ❌ 较难构建（缺乏现成数据集）

| 场景 | 难点 | 可能的替代方案 |
|------|------|----------------|
| 个人文件检索 | 隐私数据 | 模拟生成个人文档库 |
| 家庭物资/账单 | 私人数据 | 合成家庭消费数据 |
| 本地生活资源 | POI+个性化偏好 | 利用Yelp/大众点评公开数据改造 |
| 收藏内容检索 | 平台私有 | 爬取公开收藏夹数据 |
| 公司制度文档 | 企业内部 | 使用公开的员工手册模板 |
| 飞书/钉钉消息 | 企业IM隐私 | 使用Slack公开数据集替代 |
| 个人相册 | 肖像权隐私 | 使用PIPA或合成人脸数据 |
| 音乐版权库 | 版权问题 | 使用开源音乐数据集 |

### 🎯 推荐优先构建的场景（综合数据可用性+实用价值）

1. **代码/技术文档检索** ⭐⭐⭐⭐⭐ - 数据最丰富，CodeSearchNet+StackOverflow直接可用
2. **RAG/智能问答** ⭐⭐⭐⭐⭐ - RAGBench+TREC RAG评测成熟
3. **电商产品检索** ⭐⭐⭐⭐⭐ - Amazon数据集非常完整
4. **多模态PDF/文档检索** ⭐⭐⭐⭐ - MMDocIR/ViDoRe新兴热点
5. **法律合同检索** ⭐⭐⭐⭐ - ACORD/CUAD专业标注
6. **视频时刻检索** ⭐⭐⭐⭐ - Charades-STA/QVHighlights成熟
7. **Multi-hop推理** ⭐⭐⭐⭐ - MultiHop-RAG专门设计
8. **表格/NL2SQL** ⭐⭐⭐ - BIRD/TableQA可用，但真实场景覆盖不足
9. **中文多模态** ⭐⭐⭐ - RealBench/MUGE/Qilin新兴，持续完善中
10. **Deep Research** ⭐⭐⭐ - DeepSearchQA刚发布，评测难度大
