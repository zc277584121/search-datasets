# RAG & Intelligent Search Datasets

Retrieval-augmented generation and complex reasoning datasets.

## Datasets

- `ragbench/` - Large-scale RAG benchmark, 100K samples
- `multihop_rag/` - Multi-hop reasoning, 2556 queries
- `natural_questions/` - Google search questions
- `deepsearchqa/` - Deep research benchmark (Google DeepMind)
